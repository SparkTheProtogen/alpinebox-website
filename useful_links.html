This is a collection of utilities that may be useful to you as a user visiting this page
(They also may not be of any use at all, depending on who you are)

Web

Librewolf Browser (librewolf.net/)

Librewolf is a fork of Firefox that is tweaked to be more private and also removes the slop that newer Firefox versions added (Pocket, News on the start page, etc), it also comes with uBlock Origin pre-installed! :3

uMatrix (github.com/gorhill/uMatrix/)

uMatrix allows you to fine tune what domains a tab can contact therefore increasing privacy, some domains you might want to block are sentry.* domains, disguised trackers such as backstory.ebay.co.uk or improving.duckduckgo.com, cloudflare are not great for privacy I've heard either so cloudflareinsights.com is a good thing to block on sight a good strategy is to block everything and unblock domains one by one until the site functions.

OpenStreetMap (www.openstreetmap.org/)

OpenStreetMap is a digital map that is constructed from publicly available geographic data, it is essentilly Wikipedia if it was a map. OSM has a lot less tracking than Google Maps or similar products and has a strong focus on local knowledge and community.

4get search scraper (4get.ca/)

4get is a scraper that takes from other search engines, using 4get as a middleman probably increases privacy although you do need to solve a cpatcha to use the service and some banners on the main sceen are a little odd to say the least.

Windows

Winaero Tweaker (winaero.com/)

Winaero Tweaker allows you to alter hidden Windows settings (Increase verbosity of shutdown messages, Disable internet searches in Windows search, restore Win10 taskbar in Windows 11, Disable background apps, Restore the classic volume slider, etc.) and add extra functionality to existing Windows apps.

O&O ShutUp10++ (oo-software.com/en/shutup10)

ShutUp10++ is a set of utilities centered around minimising Windows telemetry and data collection, plus it allows you to to block some useless features Microsoft keeps pushing (MS Recall especially). An example of some options available are:

  • Disabling the camera in the login screen

  • Disabling and resetting Advertising IDs for the current user or whole machine

  • Opt out of MS running experiments (A/B testing) on your machine

Safing Portmaster (safing.io)

Safing Portmaster is a software that allows you to monitor what connections an application on your computer is making and block them if necessary, It does phone home to safing but this is solely to check if you have a subscription active. No telemetry otherwise :)

Microsoft Powertoys (learn.microsoft.com/en-us/windows/powertoys/install)

Microsoft PowerToys is a set of utilities that may come in handy for power users, some utilities are outlined below.

  • PowerRename - Mass rename files (Regex is supported for advanced pattern matching)

  • Mouse without borders - Use another Windows computer as if it was another monitor on your machine

  • File Locksmith - Get windows to tell you *what* program the file is open in instead of just complaining about the fact that it is when you try to delete/move it
